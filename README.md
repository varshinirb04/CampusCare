# CampusCare - Smart Complaint Tracking System

CampusCare is a responsive landing page created using **HTML5** and **Tailwind CSS**.  
It represents a campus-based complaint management system where students can raise, track, and manage campus-related issues.

## Project Overview

This project is designed as part of a Full Stack Development lab exercise.  
The main objective is to create a modern and responsive landing page using HTML5 semantic elements and Tailwind CSS utility classes.

## Features

- Responsive landing page
- Modern hero section
- Navigation menu
- Complaint workflow section
- Feature cards
- Campus complaint categories
- Footer with contact information
- Smooth scrolling
- Hover effects
- Custom CSS animations

## Technologies Used

- HTML5
- Tailwind CSS CDN
- CSS3
